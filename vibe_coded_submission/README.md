# Vibe Coded Submission

This version of the Expense Tracker was developed using AI-assisted vibe coding.

The application was created directly through prompts and iterative code generation without formal specifications.

## Features

- **Add Expense**: Easily input the expense name and amount.
- **Display Expenses**: View all added expenses in a list.
- **Delete Expenses**: Remove any expense from the list.
- **Show Total**: See the total sum of all logged expenses in real time.

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
